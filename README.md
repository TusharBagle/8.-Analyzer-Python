# NumPy Analyzer

## About
NumPy Analyzer is a Python-based application designed to perform various NumPy array operations through an interactive menu-driven interface.

This project demonstrates important concepts of numerical computing and array manipulation using the NumPy library while improving logical thinking, data handling, and problem-solving skills.

## Features
- Create 1D, 2D, and 3D NumPy arrays
- Perform array mathematical operations
- Combine and split arrays
- Search values in arrays
- Sort and filter arrays
- Compute statistical operations
- Interactive menu-driven system

## NumPy Operations Used
- Array Creation
- Array Reshaping
- Mathematical Operations
- Vertical Stacking
- Array Splitting
- Searching with np.where()
- Sorting with np.sort()
- Filtering Arrays
- Statistical Functions

## Statistics Included
- Sum
- Mean
- Median
- Standard Deviation
- Variance

## Technologies Used
- Python
- NumPy

## Purpose
The main purpose of this project is to strengthen understanding of NumPy operations, array manipulation, and numerical computing concepts through practical implementation.
